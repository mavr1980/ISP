# ISP

Приложение для расчёта теплопоступлений от солнечной радиации через ограждающие конструкции.

## Сборка и запуск

Проект написан на VB.NET (Windows Forms). Для сборки требуется .NET 6 SDK или новее. Запустить сборку можно командой:

```bash
dotnet build SolarGainApp/SolarGainApp.vbproj
```

Запуск приложения:

```bash
dotnet run --project SolarGainApp/SolarGainApp.vbproj
```

Главное окно позволяет выбрать город из списка и ввести параметры до десяти ограждающих конструкций. Результатом расчёта является график теплопоступлений для каждой конструкции и суммарный график для помещения.
