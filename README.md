# ISP

Input Solar Power

## Usage

Build the VB.NET Windows Forms application to calculate solar heat gains. Open the solution in Visual Studio or run `dotnet build` and execute the generated `SolarGainCalculator.exe`. Select a city, enter up to ten building surfaces with their orientation, area, and solar heat gain coefficient (SHGC), then click **Calculate** to view monthly heat gain charts for each surface and the total.
